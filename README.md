# Simple encrypt/decrypt message

# Install
```
git clone https://github.com/night3098/SimpleEncryptMsg
cd SimpleEncryptMsg
c++ -x c -o encrypt encrypt.c
c++ -x c -o encrypt decrypt.c
```
# Use

```
./encrypt
./decrypt
```
